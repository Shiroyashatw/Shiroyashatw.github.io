租金－【未核定及未領補貼時】原申請人死亡變更申請人
檢附資料 處理流程
  1. 租金補貼申請書(正本)※已有包含衛切
  2. 原申請人死亡證明書影本或除戶謄本(影本)
  3. 變更後申請人一個月內全戶戶籍謄本或戶口名簿(影本)
  4. 整份租賃契約書影本(影本)※承租人應為變更後申請人
  5. 申請人郵局帳號封面(影本)
  6. 新申請人及其配偶、戶籍內直系親屬一個月內財產歸屬清單(影本)
  7. 新申請人及其配偶、戶籍內直系親屬最新年度各類所得清單(影本)
  1. 先打好申請人身故的備註
  2. 更改申請人資料:名字、ID、生日/郵局帳號
  3. 評分重新載入、並點成初審符合
  4. 財稅全點符合/房屋否、數量0/按檢核
  5. 點選成複審合格
  6. 寫信給(kcl@cpami.gov.tw)，協請更改租金案件狀態
  7. 內容:案件編號/新申請人名/新申請人ID/排序為中央或地方
  8. 收到回信後，變更好排序狀態，契約查核補約
  9. 資料正本掃描上傳，系統key備註，最後掛文給承辦
注意事項
  • 繼承之申請人必須為身故申請人"申請時"戶籍內直系親屬或家庭成員
  • 核定前原申請人死亡，應由原書表家庭成員收受通知兩個月內辦理更名申請，且依規定重新審查新申請人與家庭成員資格。
租金－【已受領1期以上補貼時】原申請人死亡變更申請人
檢附資料 處理流程
  1. 租金補貼更名申請書(正本)
  2. 原申請人死亡證明書影本或除戶謄本(影本)
  3. 變更後申請人一個月內全戶戶籍謄本或戶口名簿(影本)
  4. 整份租賃契約書影本(影本)※承租人應為變更後申請人
  5. 申請人郵局帳號封面(影本)
  6. 家戶人口及衛浴設備切結書(影本)
  1. 先打好申請人身故的備註，紀錄原本的排序結果
  2. 更改申請人資料:名字、ID、生日/郵局帳號
  3. 評分重新載入、並點成初審符合
  4. 財稅全點符合/房屋否、數量0/按檢核
  5. 點選成複審合格
  6. 寫信給(kcl@cpami.gov.tw)，協請更改租金案件狀態

  8. 收到回信後，變更好排序狀態，契約查核補約

  • 核定後原申請人死亡，原書表家庭成員應於收受通知三個月內提出更名申請
    原戶籍有未成年(未滿20歲)親屬也可以繼承，核定後繼承不侷限申請時申請人資格。
租金－遷居申請
檢附資料 處理流程
  1. 租金補貼遷居申請書(正本)
  2. 原居住地主管機關核發之核定函(影本) △非必要
  3. 一個月內全戶戶籍謄本或戶口名簿(影本)
  4. 整份租賃契約書(影本)
  5. 第二類建物登記謄本或建物所有權狀(影本)
  6. 家戶人口及衛浴設備切結書(影本)
  7. 申請人及其配偶、戶籍內直系親屬一個月內財產歸屬清單(影本)
  8. 申請人及其配偶、戶籍內直系親屬最新年度各類所得清單(影本)
注意事項

原在外縣市租屋，後來搬到臺中市租屋應向臺中辦理遷居申請

審查期間遷居應以核定函三個月內至該縣市辦理遷居申請

核定後遷居應以遷居日(新租約起始日)三個月內至該縣市辦理遷居申請

超過三個月故取消補貼資格，並由原補貼縣市追回溢領。
租金－原申請時尚未租屋
檢附資料 處理流程
  1. 一個月內全戶戶籍謄本或戶口名簿(影本)
  2. 整份租賃契約書(影本)
  3. 第二類建物登記謄本或建物所有權狀(影本)
  4. 家戶人口及衛浴設備切結書(影本)
注意事項

申請人應於核定函核發後3個月內檢附以上相關資料

若承租地址與原申請時欲租縣市不同時，須辦理遷居申請。
租金－一般補約
檢附資料 處理流程
  1. 整份租賃契約書(影本)
  1. 從「查核」>「契約查核」，選擇年度及案件編號進入登錄資料頁面
  2. 依序填入租約資料並儲存
  3. 回到「案件查詢」，確認「檢附文件」下方已更新且資訊正確，複製下方備註內容貼到上方備註欄，並按修改
  4. 租約下方簽名壓日期，掃描上傳並成冊
注意事項
租金－搬家申請
檢附資料 處理流程
  1. 一個月內全戶戶籍謄本或戶口名簿(影本)
  2. 整份租賃契約書(影本)
  3. 第二類建物登記謄本或建物所有權狀(影本)
  4. 家戶人口及衛浴設備切結書(影本)
  1. 從「查核」>「契約查核」，選擇年度及案件編號進入登錄資料頁面
  2. 核對家庭人口數有無改變，面積有無符合居住水準
  3. 依序填入租約資料並儲存
  4. 若有變更戶籍、通訊地址，則回到「案件查詢」，進入「申請人基本資料」修改戶籍地及通訊地址
  5. 到「檢附文件」下方，確認已更新且資訊正確，複製下方備註內容貼到上方備註欄，並按修改
  6. 租約下方簽名壓日期，掃描上傳並成冊
注意事項

審查期間租約消滅者申請人應於核定函核發後3個月內辦理搬家手續

受補貼期間應於原租約到期或新租約起始日3個月內辦理搬家手續

一個月內戶政說法：因規定承租住宅須符合基本居住水準，故需依戶政人數判別家戶人口數。
租金－出租人變更
檢附資料 處理流程
  1. 整份租賃契約書(影本)
注意事項

辨理出租人變更，僅須提供"與新出租人簽署之合約"即可辦理

若出租人過世，系統會自動比對停止撥款
租金－變更郵局
檢附資料 處理流程
  1. 郵局變更切結書
  2. 變更人身分證資料(戶籍資料)
  3. 郵局存摺封面影本
  1. 到「檢附文件」下方，直接更改郵局帳號資料，若非本人需key受款人資料
  2. 到上方備註欄，填寫修改日期並按修改
  3. 資料下方簽名壓日期，掃描上傳並成冊
注意事項
自購－核定後購屋
檢附資料 處理流程
  1. 一個月內全戶戶籍謄本或戶口名簿(影本)
  2. 房屋所有權狀或第一類建物登記謄本(影本)
  3. 貸款餘額證明(影本)※貸款人姓名、ID、申貸日、年限、餘額、擔保品地址、銀行章、開立日期、需載明貸款用途(應為購屋用途)
  4. 家戶人口及衛浴設備切結書(影本)
注意事項
取得自購住宅貸款利息補貼證明之日起1年內，應與承貸金融機構簽訂貸款契約
自購－【核定後】原申請人死亡變更申請人
檢附資料 處理流程
  1. 自購更名申請書(正本)
  2. 原申請人死亡證明書影本或除戶謄本(影本)
  3. 變更後申請人一個月內全戶戶籍謄本或戶口名簿(影本)
  4. 房屋所有權狀或第一類建物登記謄本(影本)
  5. 貸款餘額證明(影本) ※貸款人姓名、ID、申貸日、年限、餘額、擔保品地址、銀行章、開立日期、需載明貸款用途(應為購屋用途)
  6. 家戶人口及衛浴設備切結書(影本)
注意事項

變更前，可以先請繼承人至銀行送聯徵，再請行員來電告知是否聯徵通過再進行辦理變更手續

※貸款餘額證明須等本同意更名後再檢附。

  2. 原申請人死亡證明書影本或除戶謄本(影本)
  3. 變更後申請人一個月內全戶戶籍謄本或戶口名簿(影本)
  4. 房屋所有權狀或第一類建物登記謄本(影本)
  5. 貸款餘額證明(影本) ※貸款人姓名、ID、申貸日、年限、餘額、擔保品地址、銀行章、開立日期
注意事項
修繕－急迫性修繕
檢附資料 處理流程
  1. 急迫性修繕申請書(正本)
  2. 一個月內全戶戶籍謄本或戶口名簿(影本)
  3. 房屋所有權狀或第一類建物登記謄本(影本)
  4. 火災、天災致房屋不堪居住之證明文件(影本)
  5. 修繕前照片
注意事項

辦理修繕貸款利息補貼前，房屋不可進行修繕

若有不堪居住之急迫性需求，須先提出申請，同意後才可進行修繕

提醒申請人，急迫性修繕僅只有同意先行動工，並非享有利息補貼

須於受理期間提出利息補貼申請，需請申請人注意已貸款者不得申請轉貸
